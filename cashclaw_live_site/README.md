# CashClaw Live Site

Live local website for CashClaw metrics.

## Run
```bash
cd /home/ander/.openclaw/workspace/cashclaw_live_site
node server.js
```

Open: http://localhost:4060

## What it shows
- total jobs
- applied/created
- in progress
- completed
- unpaid/paid
- total pipeline value
- total earned
- recent jobs table

Data source: `~/.cashclaw/`
